You have 4 options for the badge on this page. Pick one!

Follow these steps for the first 3 options:

  1. Turn on the badge whilst holding the bottom left hardware button to make it discoverable over Bluetooth.
  2. Release hardware button when told on screen.
  3. Click the relevant purple button below
  4. In the popup that appears, click on your device, based on Bluetooth address above
  5. Wait approx 30 seconds and it'll be done

Option 1 - Flash badge and permanently make it a clock

IMPORTANT: This will remove all the NodeConf User Apps and replace them with just the clock.

Or Option 2 - Temporarily turn the badge into a clock

This just runs the clock code from RAM and it will need to be run again if you turn the badge power off or the battery runs out.

Or Option 3 - Update the time on an existing badge-clock

If you power cycle the badge, it'll lose the time. This will send the time from your computer.

Or Option 4 - Reflash a badge-clock back to stock NodeConf code

If you previously chose Option 1 but now want to go back to the standard NodeConf code, just go here and follow the instructions.
